The logistic model

Consider the following model for a population: \[ N' = \beta(N)N - \mu(N) N, \]

where \(\beta(N)\) is the fertility rate and \(\mu(N)\) is the mortality rate. We assume here both are function of the population size \(N\) as follows: \[ \begin{aligned} \beta(N) &= \beta_0 - \tilde{\beta} N, \\ \mu(N) &= \mu_0 + \tilde{\mu} N. \end{aligned} \]

  1. Show that the model can be written as \[N' = r\Bigl(1-\frac{N}{K}\Bigr)N.\tag{1}\]
Solution

Substituting we get: \[ \begin{split} N' &= \beta(N)N - \mu(N)N = (\beta_0 - \mu_0)N - (\tilde{\mu}+\tilde{\beta})N^2 \\ &= (\beta_0 - \mu_0)\Bigl( 1 - \frac{\tilde{\mu}+\tilde{\beta}}{\beta_0 - \mu_0} N \Bigr) N \\ &= r \Bigl( 1 - \frac{N}{K} \Bigr) N, \end{split} \]

with \(r = \beta_0 - \mu_0\) and \(K = r / \tilde{\mu}+\tilde{\beta}\).

  1. Prove that the ODE, supplemented with an initial condition \(N(0)=N_0\), has a unique local solution. Hint: verify the hypotheses of the Cauchy-Lipschitz theorem.
Solution

The right hand side \[ f(t,y) = r \Bigl( 1 - \frac{N}{K} \Bigr) N \]

is a polynomial in \(y\) with no explicit dependency on \(t\). Thus, it is continuous and also locally Lipschitz. We can apply the Cauchy-Lipschitz theorem and conclude that there exists a unique local solution.

  1. Consider \(u(t) = 1/N(t)\). Show that \(u(t)\) satisfies a linear ODE. Solve it, then use the solution to solve the original ODE in \(N(t)\) with the initial condition \(N(0) = N_0 > 0\).
Solution

We have \[u'= -\frac{N'}{N^2} = -\frac{r}{N^2}\Bigl(1 - \frac{N}{K}\Bigr)N = r\Bigl(\frac{1}{K} - \frac{1}{N}\Bigr) = r\Bigl(\frac{1}{K} - u \Bigr),\]

which is linear with respect to \(u\). To solve the ODE, we can set \(\tilde{u}=u-K^{-1}\), \(\tilde{u}'=u'\), so the ODE reduces to \(\tilde{u}'= -r\tilde{u}\). The general solution is: \[u(t) = \tilde{u}(t) + \frac{1}{K} = \frac{1}{K} + C e^{-r t}.\]

Since \(u(0) = 1/N(0) = 1/N_0\), we can determine the constant: \[u(t) = \frac{1}{K} + \tilde{u}(t) = \frac{1}{K} - \Bigl(\frac{1}{N_0} - \frac{1}{K} \Bigr)e^{-rt}.\]

If we invert \(u(t)\) we obtain \[N(t) = \frac{K N_0}{N_0 + (K-N_0)e^{-rt}}\]

which is the sought solution of the logistic equation.

  1. Make the change of variables as follows: \(\tau = r t\) and \(y = N/K\). Find the corresponding equation in \(y(\tau)\).
Solution

We simply note that \[\dot{y}=\frac{\mathrm{d}y}{\mathrm{d}\tau}=\frac{1}{Kr}\frac{\mathrm{d}y}{\mathrm{d} t} = \frac{1}{Kr}r Ky(1-y) = y(1-y).\]

So the equation does not depend on the parameters anymore.

Blow-up of solutions

It may be considered reasonable that, for a sexual species, births are proportional to the number of encounters, hence, disregarding the mortality process, we have the equation \[N'(t) = \beta N^2(t).\]

  1. This equation can be solved by the method of separation of variables. Show that the solutions of this equation with \(N(0) > 0\) tend to infinity in a finite time. (Blow-up of solution.)
Solution

We can solve by separation of variables: \[\int_{N_0}^N \frac{\mathrm{d}N}{N^2}=\int_0^t \beta \mathrm{d}t,\quad\leadsto\quad N(t) = \frac{N_0}{1 - N_0 \beta t}.\]

Since \(N_0>0\) and \(\beta>0\), there is a vertical asymptote at \(\bar{t} = \frac{1}{\beta N_0}\). Hence, we have a blow-up of the solution in finite time.

  1. Let us correct the equation, introducing deaths: \[N'(t) = \beta N^2(t) − \mu N(t).\]

Discuss how the dynamics changes. In particular, does the equation still have the problem of solutions going to infinity in a finite time? Hint: do not solve analytically the equation.

Solution

Yes, it does. Consider \(N_0\) large, then \(f(N)=\beta N^2-\mu N \approx \beta N^2\) so we expect the same problem.

  1. Solve the above equation analytically and confirm the qualitative analysis of the previous point.
Solution

If we rewrite the problem as \[ N' = -\mu N (1 - \tfrac{\beta}{\mu} N), \]

we recover the logistic equation “with the wrong sign”, with \(r=-\mu\) and \(K = \tfrac{\mu}{\beta}\). We can use the solution from the previous exercise: \[N(t) = \frac{K N_0}{N_0 + (N_0-K)e^{-rt}}.\]

Since \(r = -\mu < 0\), the denominator can be zero when: \[ e^{\mu t} = \frac{N_0}{K-N_0} \ge 1, \]

This is the case when \(N_0 \ge K = \tfrac{\mu}{\beta}\).

mRNA

Zeisel et al. considered the following system for the concentration of mRNA. The variables are \(M(t)\), the concentration of mature mRNA, and \(P(t)\), the concentration of precursor mRNA: \[ \left\{\begin{aligned} P'(t) &= b(t) − \alpha_1 P(t), \\ M'(t) &= \alpha_1 P(t) − \alpha_2(t) M(t). \end{aligned}\right. \]

where \(b(t)\) is the (gene-specific and time-dependent) production rate, \(\alpha_1\) is the conversion (splicing) rate of pre-mRNA to mRNA, \(\alpha_2(t)\) the (time-dependent) degradation rate of mRNA.

  1. Assume \(b(t) \equiv b\) and \(\alpha_2(t) \equiv \alpha_2\) are constant. Find the explicit solution given \(P(0)=P_0\) and \(M(0)=M_0\). Hint: use the method of variation of constants.
Solution

We note that the first equation does not depend on \(M(t)\), so it can be solved independently. The solution is: \[ P(t) = e^{-\alpha_1 t} P_0 + \frac{b}{\alpha_1}(1-e^{-\alpha_1 t}). \]

Now we can move this into the equation for \(M(t)\): \[ M'(t) = e^{-\alpha_1 t} P_0 + b (1-e^{-\alpha_1 t}) - \alpha_2 M(t). \]

Here, we use the variation of constants. The solution of the homogeneous equation, denoted by \(M_H(t)\) is: \[ M_H(t) = C e^{-\alpha_2 t}, \]

so we look for solutions of the form: \[ M(t) = C(t) e^{-\alpha_2 t}. \]

Substituting: \[ M'(t) = C'(t) e^{-\alpha_2 t} - \alpha_2 C(t) e^{-\alpha_2 t} = C'(t) e^{-\alpha_2 t} - \alpha_2 M(t). \]

Equating this with the original equation we find an equation for \(C(t)\): \[ \begin{split} C'(t) &= \alpha_1 e^{-(\alpha_1-\alpha_2) t} P_0 + b (1-e^{-\alpha_1 t}) e^{\alpha_2 t} \\ &= b e^{\alpha_2 t} + (\alpha_1 P_0 - b) e^{-(\alpha_1-\alpha_2) t}, \end{split} \]

that integrated gives \(C(t)\) \[ C(t) = C(0) + \tfrac{b}{\alpha_2} e^{\alpha_2 t} - \tfrac{\alpha_1 P_0 - b}{\alpha_1-\alpha_2} e^{-(\alpha_1-\alpha_2) t} \]

and the final solution: \[ M(t) = C(0) e^{-\alpha_2 t} + \tfrac{b}{\alpha_2} - \tfrac{\alpha_1 P_0 - b}{\alpha_1-\alpha_2} e^{-\alpha_1 t}. \]

The integration constant \(C(0)\) can be selected so that \(M(0)=M_0\): \[ C(0) = M_0 - \tfrac{b}{\alpha_2} + \tfrac{\alpha_1 P_0 - b}{\alpha_1-\alpha_2}. \]

  1. Assume now that \(\alpha_2(t) \equiv \alpha_2\) and \[ b(t) = \begin{cases} \bar{b} & t < \bar{t}, \\ 0, & t > \bar{t}. \end{cases} \]

Show that \(M(t)\) tends to \(0\) as \(t\to\infty\). At which rate does it decay?

Solution

Note that \(b(t)\) is not continuous, so we cannot expect a classical solution. However, we can proceed step by step. Informally, we can integrate up to \(t = \bar{t}^-\) with \(b(t)=\bar{b}\). The solution \(P(\bar{t})\) can then be used for a new problem with \(P_0 = \bar{P} = P(\bar{t})\) and \(b(t) = 0\). The solution for \(P(t)\) will therefore decay as \(e^{-\alpha_1 t}\). For \(M(t)\), we use the above formula, which holds true for \(t > \bar{t}\), setting \(b=0\) and \(P_0 = \bar{P}\): \[ M(t) = C(0) e^{-\alpha_2 t} - \tfrac{\alpha_1 \bar{P}}{\alpha_1-\alpha_2} e^{-\alpha_1 t}. \]

We see that \(M(t)\to 0\), no matter of the value of the constants. However, if \(\alpha_1 > \alpha_2\), the second term \(e^{-\alpha_1 t}\) will vanish quickier than the first one, so \(M(t) \sim e^{-\alpha_2 t}\) for large \(t\). In general \[ M(t) \sim e^{-\min\{ \alpha_1, \alpha_2 \}t}. \]

Alternatively, we can do all the computations. First, we integrate up to \(t = \bar{t}\), with \(b(t) = \bar{b}\), the following problem (we neglect \(M(t)\) for now): \[ \left\{ \begin{aligned} P_1'(t) &= \bar{b} - \alpha_1 P_1(t), \\ P_1(0) &= P_0, \end{aligned}\right. \]

which has the solution \[ P_1(t) = e^{-\alpha_1 t}P_0 + \frac{\bar{b}}{\alpha_1}(1-e^{-\alpha_1 t}). \]

for \(0 \le t < \bar{t}\). Then, for \(t > \bar{t}\), we solve another problem with \(b(t) = 0\): \[ \left\{ \begin{aligned} P_2'(t) &= 0 - \alpha_1 P_2(t), \\ P_2(\bar{t}) &= P_1(\bar{t}), \end{aligned}\right. \]

which has the solution \[ P_2(t) = e^{-\alpha_1(t-\bar{t})} P_1(\bar{t}) = e^{-\alpha_1 t} \Bigl[ P_0 + \frac{\bar{b}}{\alpha_1} (e^{\alpha_1 \bar{t}} - 1 ) \Bigr]. \]

The overall solution is (which is continuous at \(t=\bar{t}\) but not differentiable): \[ P(t) = \begin{cases} P_1(t), & t \le \bar{t}, \\ P_2(t), & t > \bar{t}. \end{cases} \]

Clearly, \(P(t)\to 0\) as \(t\to\infty\), and it decays as \(e^{-\alpha_1 t}\). We can use this information for \(M(t)\). In general: \[ M(t) = e^{-\alpha_2 t} M_0 + \alpha_1 \int_0^t e^{-\alpha_2(t-s)} P(s)\,\mathrm{d}s. \]

Since we are interested in large \(t\), we can start from \(\bar{t}\) instead of \(t=0\). So \(P(t) = P_2(t) = c e^{-\alpha_1 t}\): \[ \begin{split} M(t) &= e^{-\alpha_2 t} M(\bar{t}) + c \alpha_1 \int_{\bar{t}}^t e^{-\alpha_2(t-s)} e^{-\alpha_1 s}\,\mathrm{d}s \\ &= e^{-\alpha_2 t} M(\bar{t}) + c \alpha_1 e^{-\alpha_2 t} \int_{\bar{t}}^t e^{(\alpha_2-\alpha_1) s} \,\mathrm{d}s \\ &= e^{-\alpha_2 t} M(\bar{t}) + c \alpha_1 e^{-\alpha_2 t} \Bigl( \frac{1}{\alpha_2-\alpha_1} e^{(\alpha_2-\alpha_1) t} - \tilde{c} \Bigr) \\ &= e^{-\alpha_2 t} \hat{c} + \frac{\alpha_1}{\alpha_2-\alpha_1} e^{-\alpha_1 t}. \end{split} \]

Thorium-Uranium dating

The thorium-uranium method for dating rocks is based on the fact that Uranium-234 decays into Thorium-230 which in turn decays into other elements. Set \(t = 0\) the rock formation time and denoting \(U(t)\) (resp. \(T(t)\)) the amount of Uranium-234 (resp. Thorium-230) in the rock at time \(t\) (measured in years), the following differential equation system is written: \[ \left\{\begin{aligned} U'(t) &= -a U(t), \\ T'(t) &= a U(t) - b T(t), \\ U(0) &= U_0, \\ T(0) &= 0, \end{aligned}\right. \]

where \(a \approx 5.9\cdot 10^{-6} \:\text{years}^{-1}\), \(b \approx 1.9\cdot 10^{-5} \:\text{years}^{-1}\), \(U_0\) represents the initial (generally unknown) amount of Uranium-234. Note that, based on geological principles, it is believed that there was no thorium at the time of rock formation.

  1. Solve the equation for \(U(t)\).
Solution

We have \(U(t) = e^{-a t} U_0\).

  1. How can the quantities of \(a\) and \(b\) be interpreted? From the data provided can we infer the half-life of Uranium-234 and Thorium-230?
Solution

The half-life is the time \(t_{1/2}\) that takes to have \(U(t_{1/2}) = \frac{1}{2} U_0\). From the above solution we find \(t_{1/2} = \frac{\log 2}{a}\), so \(\frac{1}{a}\) is proportional to the half-life. Actually, \(\frac{1}{a}\) is the time that takes to \(U_0\) to reach \(\frac{1}{e}U_0 \approx 0.36 U_0\). So, the half-life of Uranium-234 is \(\approx 117\,500\) years, whereas for Thorium-230 (in absence of uranium) is \(\approx 36\,500\) years.

  1. Calculate \(T(t)\), solution of the second differential equation.
Solution

We substitute the solution \(U(t)\) into the second equation, so we need to solve: \[ T' = a e^{-a t} U_0 - b T, \]

with \(T(0)=0\). This equation is linear but non-homogeneous. The Wronskian “matrix” is \(e^{-bT}\), so we have \[ \begin{split} T(t) &= e^{-b T}T(0) + \int_0^t e^{-b(t-s)} a e^{-a s} U_0\,\mathrm{d}s \\ &= a U_0 e^{-bt} \int_0^t e^{(b-a) s} \,\mathrm{d}s \\ &= \frac{a U_0}{b-a} e^{-bt} \Bigl( e^{(b-a) t} - 1 \Bigr) \\ &= \frac{a U_0}{b-a} \Bigl( e^{-a t} - e^{-b t} \Bigr). \end{split} \]

  1. Compute \[ \lim_{t\to\infty}\frac{T(t)}{U(t)}. \]
Solution

We have the full solution, so we can easily compute it: \[ \frac{T(t)}{U(t)} = \frac{\frac{a U_0}{b-a}(e^{-a t} - e^{-b t})}{e^{-a t} U_0} = \frac{a}{b-a}\bigl(1 - e^{-(b-a)t}\bigr) \to \frac{a}{b-a}. \]

Note that we could have proceeded differently, by observing that: \[ \left(\frac{T(t)}{U(t)}\right)' = \frac{T'(t)}{U(t)} - \frac{T(t)}{U^2(t)}U'(t) = a - (b-a)\frac{T(t)}{U(t)}. \]

The differential equation for \(w(t) = T(t)/U(t)\) is: \[ w' = a - (b-a)w. \]

The solution is the same as above.

  1. Explain why it is possible to estimate the rock age from the knowledge of \(T / U\) at current time, but it is not possible from the knowledge of \(T\) alone. Hint: study the function \(T(t)/U(t)\).
Solution

The function \(w(t) = T(t)/U(t)\) is monotonically increasing, bounded from above by \(\frac{a}{b-a}\). It does not depend on the initial concentration \(U_0\). Thus, it is always possible to find a unique time \(\tilde{t}\) such that \[ w(\tilde{t}) = \frac{T(\tilde{t})}{U(\tilde{t})} = \tilde{w}, \]

where \(\tilde{w}\) is the thorium-uranium ratio at time \(\tilde{t}\). In particular: \[ \tilde{t} = \frac{1}{b}\ln\Bigl( \frac{a}{a-(b-a)\tilde{w}} \Bigr). \]

We cannot use \(T(t)\), because it is not a monotone function. Thus, given \(\tilde{T}\), we cannot find a unique \(\tilde{t}\) such that \(T(\tilde{t}) = \tilde{T}\).
